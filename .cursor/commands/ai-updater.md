# AI Updater

Find and upgrade all AI-related open-source projects on this machine.

## Instructions

Run the scanner to discover all AI projects, compare versions, and let the user pick which to upgrade.

```bash
python ai_updater.py
```

Present the summary table. Ask the user which to upgrade (`all`, `p`, `d`, `1,3,5`, or `q`).
